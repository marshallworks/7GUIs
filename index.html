7GUIs

7GUIs (7 Graphical User Interfaces) is a programming benchmark to compare implementations in terms of their notation. 7GUIs defines 7 tasks that represent typical challenges in GUI programming. This implements the 7 tasks using vanilla HTML, CSS & JavaScript with the goal of being as minimal as possible.

  1. Counter
  2. Temperature Converter
  3. Flight Booker
  4. Timer
  5. CRUD
  6. Circle Drawer
  7. Cells
